Tentang Saya

Saya adalah seorang mahasiswa yang memiliki minat dalam pengembangan web dan teknologi informasi. Saya telah belajar berbagai bahasa pemrograman dan alat pengembangan seperti HTML, CSS, JavaScript, dan PHP. Saya juga memiliki pengalaman dalam proses pengembangan web sederhana, mulai dari perencanaan hingga implementasi. Selain itu, saya memiliki kemampuan dalam bekerja dengan tim dan berkomunikasi dengan baik.

Dibalik minat saya pada teknologi, saya juga tertarik pada dunia otomotif dan sepak bola, di dunia sepak bola saya mendukung tim terbaik di London saat ini yaitu CHELSEA.
